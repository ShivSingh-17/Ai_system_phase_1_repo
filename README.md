# AI Camera System 

Real-time AI-powered camera system using YOLOv8 and OpenCV for people detection and crowd analysis.

## Features
- Real-time person detection
- Unique ID tracking
- Crowd density calculation
- FPS monitoring
- Grid-based crowd heatmap logic

## Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8
- NumPy
